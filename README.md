# ☕ Java & Web Development Fundamentals

## 📌 About this Repository
This repository contains the source code, exercises, and architectural concepts developed during the **Java and Spring Boot Formation** by AlgaWorks. 

As a Global Solutions Architect with over 25 years of experience in complex corporate ERP implementations, this repository serves as my technical foundation in modern software engineering. The goal is to bridge deep business domain maturity with hands-on Backend Development, mastering the Java ecosystem from basic syntax to object-oriented design and web applications.

## 📚 Syllabus & Core Concepts Covered

This learning path is structured into 14 comprehensive modules, covering the core principles of the Java language and algorithm design:

1. **Introduction:** Java ecosystem, JVM, JRE, and JDK concepts.
2. **Getting Started:** Development environment setup and IDE tooling.
3. **Variables & Constants:** Data types, memory allocation, and best practices.
4. **Operators:** Arithmetic, logical, and relational operators.
5. **Control Flow:** Decision structures (if/else, switch).
6. **Iteration:** Loop structures (for, while, do-while).
7. **Arrays (Vectors):** Data structures, memory management, and iteration over collections.
8. **Methods:** Function definitions, parameters, return types, and scope.
9. **Object-Oriented Programming (OOP):** Classes, objects, encapsulation, inheritance, and polymorphism.
10. **File I/O:** Reading and writing data in files using Java IO/NIO.
11. **Dependency Management:** Utilizing third-party libraries and understanding package management.
12. **Advanced Algorithms:** Logic building, sorting, and performance optimization.
13. **Web Fundamentals:** Prerequisites and architecture for building web applications.
14. **First Web Application:** Building and deploying the foundational web project.
